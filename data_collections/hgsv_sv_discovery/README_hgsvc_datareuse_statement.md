##Data Reuse policy

This README relates to all data in or under this directory.

In April 2019, this work was published. The publication, [Chaisson et al. 2019 Nature Communications](https://www.nature.com/articles/s41467-018-08148-z), should be cited if use is made of this data. 

Please email info@1000genomes.org with any questions about either the index files or the data reuse policy.
